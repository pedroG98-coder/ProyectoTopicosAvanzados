Práctica:"Registro de Usuarios(SQL Server)"

Solución de la Actividad 5

• Creamos un nuevo proyecto de Android Studio.

• Seleccionamos la plantilla Emty Activity.

• Le damos un nombre y finish.

• Colocaremos los elementos correspondientes.

• 2 TextView.

• 2 EditText.

• 1 Button.

• Modificaremos su texto e ID.

• Quedará de la siguiente manera.

• Creamos una base de datos sencilla en SQLServer y agregamos unos datos.

• Ahora nos dirigiremos a la carpeta manifest y abriremos el archivo AndroidManifest.xml.

• Creamos los permisos a internet.

• Quedará de la siguiente manera.

• En este caso usaremos un Driver de java de conexión, el cual nos permitirá conectarnos con la base de datos.

• Nos dirigimos a cambiar la vista de nuestro proyecto como se muestra a continuación y colocaremos el Dirver en la carpeta app, libs.

• Para descargar la librería tendrás que ingresar a este sitio web: Java Driver Connectionn le darás a download y lo colocaras en la carpeta libs como ya lo he mencionado.

• Una vez agregada le daremos clic derecho al Driver y elijaremos Add ass Library.

• Ahora nos dirigiremos a la clase MainActivity y colocaremos los objetos tipo EditText y un Button.

• Y los asociaremos los elementos de nuestro layout con dichos objetos.

• Crearemos una función la cual permitirá establecer conexión con la base de datos a la queramos.

• Agregamos un objeto de la clase conexión para crear una conexión con la base de datos.

• Luego registramos el permiso de acceso para la aplicación.

• Conectaremos la aplicación al SQLServer.

• Usamos un Class.forname para declarar el driver de conexión.

• Obtenemos la conexión a la base de datos que deseamos, como argumentos colocaremos la dirección ip de nuestro pc, en nombre de la base de datos, usuario y contraseña.

• Quedará de la siguiente manera:

• Ahora crearemos un método el cual se encargara de agregar los datos a la base de datos.

• Haciendo uso de la clase PreparedStatement y la función creada, hacemos uso de una consulta insert con sus parámetros requeridos.

• Luego asignamos los valores a los parámetros, en los dos EditText de nuestro formulario.

• Mediante executeUpdate ejecutamos la consulta creada.

• Por último, colocamos un mensaje de confirmación y uno en caso de error.

• Nos ubicamos en el método onCreate del MainActivity.

• Crearemos el evento clic de nuestro botón.

• Dentro del evento colocamos el método agregarUsuario.

• Solo queda probar la aplicación y observar cómo se van registrando los elementos a la base de datos.
