Práctica:"Ordenar No Aleatorios de manera Ascendente y Descendente"

Solución de la Actividad 5

• Para poder realizar esta práctica deveras crear un proyecto en visual studio, utilizar la plantilla de biblioteca de clases (.NET Framework). Le darás nombre y crearas el proyecto.

• Por buenas prácticas es recomendable cambiar el nombre a la clase generada por un nombre que valla acorde a la práctica que se esté realizando. Ahora dentro de la clase colocaremos el síguete código. 

                    
public List ordenarDescendente(List listaNumeros)
    {
        List ordenados = listaNumeros;
        ordenados.Sort((a, b) => b.CompareTo(a));
        return ordenados;
    }
    public List ordenarAscendente(List listaNumeros)
    {
        List ordenados = listaNumeros;
        ordenados.Sort((a, b) => a.CompareTo(b));
        return ordenados;
    }

• Lo que hace el código es crear una lista de números el cual los compara uno por uno, ya sea ascendente o descendente y retorna una lista con los nueros ordenados. Después de colocar el código daremos clic derecho sobre el proyecto y seleccionaremos la opción de compilar para poder generar la dll necesaria para usarla en nuestro síguete programa. Recuerda que para encontrar la ruta de la dll creada tendrás que dar clic derecho sobre el proyecto y seleccionar Abrir carpeta en el Explorador de archivos, una vez abierta la ventana, abrirás la carpeta bin y después debug, ahí estará la dll creada al momento de compilar el programa.

• Guarda la ruta, ya que se utilizará para poder utilizarla en el siguiente programa.

• Ahora crearemos el programa que utilizara la dll creada, crearemos un proyecto de consola. Como se muestra a continuación:

• Le daremos a síguete, colocaremos un nombre al proyecto y daremos a crear. Una vez creado el proyecto daremos clic derecho en referencias y seleccionaremos agregar referencia.

• Ahora colocaremos el síguete código para poder utilizar los métodos de la dll.
