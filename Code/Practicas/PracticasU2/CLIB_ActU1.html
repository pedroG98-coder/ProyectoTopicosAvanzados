Práctica:"Mini Calculadora"

Solución de la Actividad 1

Para comenzar con la actividad debemos:

• Abrir nuestro IDE de Visual Studio 2019.

• Crear el nuevo proyecto.

• Escoger la plantilla de “Aplicación de Windows Forms (.Net Framework).”

• Dar clic en siguiente.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Agregaremos un nombre al proyecto.

• Agregaremos un nombre a la solución o esta a su vez puede ser la misma que la del proyecto.

• Clic en crear.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Posteriormente nos dirigimos a la pestaña que dice “cuadro de herramientas”.

• Arrastramos cuatro “Labels” hacia el formulario para las Etiquetas.

• Arrastramos dos “Textbox” hacia el formulario.

• Arrastramos cuatro “Botones” hacia el formulario.

• Como se muestra a continuación.

• Después le damos clic derecho a la solución, el cual nos abrirá una ventana de opciones.

• Escogemos la de agregar y después la de nuestro proyecto.

• Como se muestra a continuación.

• Se crea el nuevo proyecto.

• Escoger la plantilla de “Biblioteca de Clases (.Net Framework).”.

• Dar clic en siguiente.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Enseguida vemos en nuestra solución que tenemos dos proyectos, uno consta del formulario y el otro es de la DLL.

• Se crea un archivo llamado “Class1.cs” pero yo lo renombre y le puse “OBasicas.cs” para que sea más entendible.

• En el archivo “OBásicas” es donde crearemos nuestras funciones para las operaciones básicas de sumar, restar, multiplicar y dividir, para así posteriormente utilizarlos en nuestro Formulario principal que en este caso sería en el de” MiniCalculadoraDll”.

• Después de abrir el Archivo “OBasicas.cs” pondremos el siguiente código.

• Crearemos una función estática de tipo flotante “Sumar” que recibe como parámetro dos objectos del mismo tipo.

• Se retorna el valor que daremos en este caso es sumar seria “a + b”.

• Se repite el procedimiento para las siguientes operaciones (Resta, Multiplicación y División).

• Ahora incorporaremos nuestro archivo “OBasicas.cs” para nuestro formulario.

• En el proyecto “MiniCalculadoraDll” daremos clic derecho a la opción de “Referencias”.

• Escogemos la opción de “Agregar Referencia”.

• Enseguida se abrirá una nueva interfaz como la siguiente:.

• Vemos que este habilitado la opción de “Proyectos”.

• Palomeamos el checkbox de “OperacionesBasicas”.

•Verificamos que se allá agregado correctamente nuestro archivo en “Referencias”, como se muestra a continuación.

• Daremos clic derecho sobre el formulario y seleccionaremos ver código o en su caso daremos F7.

• Agregaremos la librería que creamos.

• Using “OperacionesBasicas” esto es para que nos permite ejecutar correctamente las operaciones.

• A continuación, le daremos clic al botón” Sumar” esto para que nos cree un método clic.

• Creamos una excepción.

• Mandamos a traer el Label donde mostraremos el resultado, de igual manera la librería que creamos anteriormente con la función que deseamos obtener en este caso seria “Sumar”, interpretamos las cajas de textos en tipo flotante esto para que nos acepte valores con decimales y por último lo pasamos a ToString para que se muestre en el label.

• En el catch de la excepción mandamos un mensaje en el label.
