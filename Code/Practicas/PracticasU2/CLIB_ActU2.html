Práctica:"Ecuación Cuadrática"

Solución de la Actividad 2

• Para comenzar con la actividad debemos:

• Abrir nuestro IDE de Visual Studio 2019.

• Crear el nuevo proyecto.

• Escoger la plantilla de “Aplicación de Windows Forms (.Net Framework).”

• Dar clic en siguiente.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Agregaremos un nombre al proyecto.

• Agregaremos un nombre a la solución o esta a su vez puede ser la misma que la del proyecto.

• Clic en crear.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Posteriormente nos dirigimos a la pestaña que dice “cuadro de herramientas”.

• Arrastramos ocho “Labels” hacia el formulario para las Etiquetas.

• Arrastramos cinco “TextBox” hacia el formulario.

• Arrastramos un “Botón” hacia el formulario.

• Como se muestra a continuación.

• Después le damos clic derecho a la solución, el cual nos abrirá una ventana de opciones.

• Escogemos la de agregar y después la de nuestro proyecto.

• Como se muestra a continuación.

• Se crea el nuevo proyecto.

• Escoger la plantilla de “Biblioteca de Clases (.Net Framework).”.

• Dar clic en siguiente.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Enseguida vemos en nuestra solución que tenemos dos proyectos, uno consta del formulario y el otro es de la DLL.

• Se crea un archivo llamado “Class1.cs” pero yo lo renombre y le puse “Operaciones.cs” para que sea más entendible.

• En el archivo “Operaciones.cs” es donde crearemos nuestras funciones para las operaciones básicas de sumar, restar, multiplicar y dividir, para así posteriormente utilizarlos en nuestro Formulario principal que en este caso sería en el de “EcuacionCuadraticaDLL”.

• Después de abrir el Archivo “Operaciones.cs” pondremos el siguiente código.

• Crearemos una función pública de tipo double “x1” que recibe como parámetro a, b, c de tipo entero.

• Se crea una variable “Res” el cual será nuestro valor retornado se inicializa en 0.

• Se crea una variable “Pot” el cual será la potencia y de parámetro mandamos “B”.

• Evaluamos lo que está dentro de nuestra raíz cuadrada, mediante la fórmula Gral.

• Evaluamos con un If para solo utilizar valores reales.

• Sacamos la raíz cuadrada de Z.

• En la variable “Res” evaluamos mediante la fórmula general.

• Por últimos retornamos “Res”, se repite la misma el procedimiento, pero ahora en vez de “Menos”, será “Mas” raíz, como se muestra a continuación.

• Ahora incorporaremos nuestro archivo “Operaciones.cs” para nuestro formulario.

• En el proyecto “EcuaciónCuadráticaDLL” daremos clic derecho a la opción de “Referencias”.

• Escogemos la opción de “Agregar Referencia”.

• Enseguida se abrirá una nueva interfaz como la siguiente:.

• Vemos que este habilitado la opción de “Proyectos”.

• Palomeamos el checkbox de “FormulaGral”.

• Verificamos que se allá agregado correctamente nuestro archivo en “Referencias”, como se muestra a continuación.

• Daremos clic derecho sobre el formulario y seleccionaremos ver código o en su caso daremos F7.

• Agregaremos la librería que creamos.

• Using “FormulaGral” esto es para que nos permite ejecutar correctamente las operaciones para el cálculo de la ecuación cuadrática.

• A continuación, le daremos clic al botón “Calcular” esto para que nos cree un método clic.

• Instanciamos operaciones de nuestra DLL.

• Obtenemos valores de tipo entero, los convertimos a Int 32 bits y se los mandamos a nuestros “TexBoxs” creados.

• Obtenemos el valor de x1 mediante operaciones y mandamos como parámetros (a, b, c), este proceso se repite para el obtener el valor de x2.

• Convertimos a ToString los TextBox, retornamos los valores de (X1 y X2) para que nos retorne el resultado.

• Por último, hacemos una condición que, si los valores de x1 y x2 equivalen a 0, nos muestre un mensaje, que no se puede resolver la ecuación, como se muestra a continuación.
