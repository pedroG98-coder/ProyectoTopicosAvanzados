Práctica:"Números Aleatorios"

Solución de la Actividad 3

Para comenzar con la actividad debemos:

• Abrir nuestro IDE de Visual Studio 2019.

• Crear el nuevo proyecto.

• Escoger la plantilla de “Aplicación de Windows Forms (.Net Framework).”

• Dar clic en siguiente.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Agregaremos un nombre al proyecto.

• Agregaremos un nombre a la solución o esta a su vez puede ser la misma que la del proyecto.

• Clic en crear.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Posteriormente nos dirigimos a la pestaña que dice “cuadro de herramientas”.

• Arrastramos un “Label” hacia el formulario para las Etiquetas.

• Arrastramos un “TextBox” hacia el formulario.

• Arrastramos dos “Botones” hacia el formulario.

• Como se muestra a continuación.

• Después le damos clic derecho a la solución, el cual nos abrirá una ventana de opciones.

• Escogemos la de agregar y después la de nuestro proyecto.

• Como se muestra a continuación.

• Se crea el nuevo proyecto.

• Escoger la plantilla de “Biblioteca de Clases (.Net Framework)”.

• Dar clic en siguiente.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Enseguida vemos en nuestra solución que tenemos dos proyectos, uno consta del formulario y el otro es de la DLL.

• Se crea un archivo llamado “Class1.cs” pero yo lo renombre y le puse “NumAleatorios.cs” para que sea más entendible.

• En el archivo “NumAleatorios.cs” es donde crearemos nuestras funciones y métodos para insertar y eliminar los números aleatorios para así posteriormente utilizarlos en nuestro Formulario principal que en este caso sería en el de “Numeros_Aleatorios”.

• Después de abrir el Archivo “NumAleatorios.cs” pondremos el siguiente código.

• Declaramos un “arreglo” y dos variables “atrás y frente” tipo byte.

• Creamos una función de números aleatorios donde el arreglo será de 5 números, e inicializamos las variables “frente y atrás” en 0.

• Creamos una función de tipo booleana cola llena, esto es para que nos diga el largo del atrás del arreglo.

• Creamos una función de tipo booleana Cola Vacía, esto es para que compare el frente con el atrás del arreglo de números.

• Creamos un método “Insertar” si la cola llena está vacía, pasara a insertar el numero Random del 1 al 100 del arreglo de 5 y aumenta la variable atrás.

• A continuación, se crea otro método, pero ahora el de eliminar si la cola vacía es igual a falso nos retornara un 0, si no pues se ejecuta lo que está dentro del if y podrá eliminar un número del arreglo y la variable atrás empieza a disminuir.

• Por último, creamos un método ToString, mientras el frente sea mayor que la variable atrás “I” aumentara y se retorna la cadena.

• Como se muestra a continuación.

• Ahora incorporaremos nuestro archivo “NumAleatorios.cs” para nuestro formulario.

• En el proyecto “Numeros_Aleatorios” daremos clic derecho a la opción de “Referencias”.

• Escogemos la opción de “Agregar Referencia”.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Vemos que este habilitado la opción de “Proyectos”.

• Palomeamos el checkbox de “DLL-NumAleatorios”.

• Verificamos que se allá agregado correctamente nuestro archivo en “Referencias”, como se muestra a continuación.

• Daremos clic derecho sobre el formulario y seleccionaremos ver código o en su caso daremos F7.

• Agregaremos la librería que creamos.

• Using “DLL_NumAleatorios” esto es para que nos permita ejecutar la Dll en el formulario.

• A continuación, traemos nuestra clase “NumAleatorios” y le damos una variable “n”, la inicializamos en el form1.

• Daremos clic al botón “Insertar” esto para que nos cree un método clic.

• Ponemos nuestra variable antes creada “n” la instanciamos con el método “Insertar” que se creó en la librería, y toda la variable n la convertimos en ToString.

• Daremos clic al botón “Eliminar” esto para que nos cree un método clic.

• Por último, ponemos nuevamente variable antes creada “n” la instanciamos con el método “Eliminar” que se creó en la librería, y toda la variable n la convertimos en ToString.
