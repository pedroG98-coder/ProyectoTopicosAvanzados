Práctica:"Archivo de Texto"

Solución de la Actividad 4

• Para poder realizar esta actividad tendrás que crear un nuevo proyecto en visual studio y seleccionaras la plantilla llamada Biblioteca de clases (.NET Framework). Como se muestra a continuación.

• Le daremos a siguiente, colocaremos un nombre al proyecto y daremos a crear. Una vez tengas el proyecto creado cambiaras el nombre de la clase generada automáticamente por Escritura, como se muestra a continuación:

• Dentro de la clase colocaremos el siguiente código:

                
public string Ruta { get; set; } 
public Escritura(string ruta)
        {
            this.Ruta = ruta;
        }
            public void Escribir(string mensaje)
        {
            StreamWriter sw = new StreamWriter(this.Ruta,true);
            sw.WriteLine(mensaje);
            sw.Close();
     }

• El cual creamos una variable llamada ruta y la encapsulamos con los métodos get y set, después creamos un método llamado Escribir el cual recibe como variable a mensaje, lo que hace es guardar el mensaje dentro de la ruta la cual tendrá un archivo de texto donde se guardará lo que hayamos escrito.

• Ahora crearemos una clase llamada Lectura y dentro de ella colocaremos el siguiente código:

                    
public string Ruta { get; set; }
public Lectura(string Ruta)
    {
        this.Ruta = Ruta;
    }
        public string Leer()
    {
        StreamReader sr = new
        StreamReader(this.Ruta);
        string contenido = sr.ReadToEnd();
        sr.Close();
        return contenido;
}

• Es similar a la clase escritura solo que en esta mostrara todo lo que se haya guardo dentro del archivo de texto.

• Ahora solo queda dar clic derecho sobre el proyecto y seleccionar compilar, esto se hace para que se genere la dll que utilizaremos en nuestro síguete programa. Para verificar que se haya creado la dll darás clic derecho sobre el proyecto y seleccionaras “Abrir carpeta desde el Explorador de archivos.”.

• Una vez abierta la ventana, abrirás la carpeta bin y después debug, ahí estará la DLL creada al momento de compilar el programa.

• Guarda la ruta, ya que se utilizará para poder utilizarla en el siguiente programa.

• Ahora crearemos el programa que utilizara la DLL creada, crearemos un proyecto de consola. Como se muestra a continuación:

• Le daremos a síguete, colocaremos un nombre al proyecto y daremos a crear. Una vez creado el proyecto daremos clic derecho en referencias y seleccionaremos agregar referencia se abrirá una ventana como esta:

• Daremos clic en examinar y colocaremos la dirección donde se encuentra guardada la dll y le daremos a agregar y aceptar.

• Colocaremos el síguete código dentro de la clase Program.cs:

                        
class Program
    {
        static void Main(string[] args)
        {
            string ruta = @"C:\pruebas\archivodll.txt";
            Escritura e = new Escritura(ruta);
            Console.WriteLine("Escriba un mensaje");
            string mensaje = Console.ReadLine();
            e.Escribir(mensaje);
            Console.WriteLine("Escrito... ");
            Console.WriteLine("");
            Console.WriteLine("Lectura... ");
            Lectura l = new Lectura(ruta);
            Console.WriteLine(l.Leer());
            Console.ReadKey();
    }
}

• Lo que hace es crear la variable ruta donde se encuentra el archivo de texto, después solo mandamos a llamar los métodos creados dentro de la DLL y los mostramos a través de una línea de comando.
