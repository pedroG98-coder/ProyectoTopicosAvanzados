Práctica:"Cuenta Numérica y Creación de Botones."

Solución de la Actividad 1

• Tendremos que crear un nuevo proyecto de visual studio, escogemos la plantilla de “Aplicación de Windows Forms (.Net Framework).” Una vez creado colocaremos los elementos necesarios para llevar a cabo esta práctica, los cuales son:

• 3 Buttons.

• 1 Listbox.

• 1 FlowLayoutPanel.

• 1 FlowLayoutPanel.

• Recuerda que para agregar estos componentes tendrás que dirigirte al apartado de “Cuadro de herramientas.”

• Como se muestra a continuación.

• Agrégale un nombre a cada botón. Tendrá que quedar de la siguiente manera:

• Ahora procederemos a programar cada uno de los componentes para que puedan realizar las acciones solicitadas, daremos clic derecho sobre el formulario y seleccionaremos ver código o en su caso daremos F7.

• Colocaremos en la parte de arriba using System.Threading; para poder crear hilos y usarlos dentro de nuestro programa, también colocaremos en el apartado del Form la siguiente propiedad, la cual evitara que mande un error al momento de inicializar más de dos hilos. Se vera de la siguiente manera:

• Ahora daremos clic al botón asignado para la creación de la cuenta numérica y colocaremos el siguiente código:

                    
Thread t = new Thread(ciclo);
t.Start();

• Lo que hará dicho código es crear un nuevo hilo que realizara un método llamado ciclo y después lo iniciamos con el comando t.start();

• Crearemos el método ciclo para la generación de la cuenta numérica, colocaremos el siguiente código:

                    
private void ciclo()
{
        for (long i = 0; i < 50; i++)
        {
            listBox1.Items.Add(i);
            Thread.Sleep(500);
        }
            MessageBox.Show("ufff.... 50");
    }

• El cual será un ciclo for que ira del 1 al 50 y lo mostrara dentro del componente listbox, este ciclo tendrá un tiempo de espera, el cual lo colocamos con el la línea de código thread.sleep(500); esto hará que la cuenta numérica tenga un pequeño descanso para mostrar el siguiente número del ciclo, también mostrara un mensaje cuando la cuenta llegue a 50. .

• Ahora daremos clic al botón encargado de mostrar el mensaje y colocaremos el síguete código:

                            
MessageBox.Show("Hola");

• Lo único que hará es mostrar el mensaje.

• Por último, daremos clic en el botón asignado para crear nuevos botones. Colocaremos el siguiente código:

                
Button CB = new Button();
        CB.Text = "#" + Cont++;
        CB.Click += click;
        this.flowLayoutPanel1.Controls.Add(CB);
        }

• Lo que hace es crear un nuevo objeto llamado CB al cual colocara como nombre a los botones #0, #1, #2, #3, etc., a como le den clic al botón, después asignamos un evento click y por último mostramos los resultados dentro del flowLayautPanel.

• El evento click deberá tener el siguiente código:

                    
Thread T = new Thread(llenar);
T.Start(sender);

• Solo estaremos creando un nuevo hilo el cual tendrá un método llamado llenar y también estaremos inicializando el hilo.

• Código del método llenar:

                        
private void llenar(object sender)
        {
            Button CB = new Button();
            CB = (Button)sender;
            for (int i = 10; i>= 0; i--)
        {
            Thread.Sleep(500);
            CB.Text = "" + i;
            if (i== 0)
        {
            CB.Enabled = false;
        }
    }
}

• Lo que hará este método es crear un objeto llamado CB y tendrá dentro un for que iniciará una cuenta regresiva del 10 al 0, el cual lo ira mostrando como texto en el botón y al momento de llegar a 0 el botón se inhabilitará.

• Ahora solo es cuestión de ejecutar el programa y observar cómo cada que presionas un botón el programa no mandara ningún error ya que cada uno funciona con un hilo diferente.
