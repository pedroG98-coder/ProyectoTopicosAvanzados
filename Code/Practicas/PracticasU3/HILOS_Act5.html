Práctica:"Dibujo de Círculos"

Solución de la Actividad 5

• Para comenzar con la actividad debemos:

• Abrir nuestro IDE de Visual Studio 2019.

• Crear el nuevo proyecto..

• Escoger la plantilla de “Aplicación de Windows Forms (.Net Framework).”.

• Dar clic en siguiente.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Agregaremos un nombre al proyecto.

• Agregaremos un nombre a la solución o esta a su vez puede ser la misma que la del proyecto.

• Clic en crear.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Posteriormente nos dirigimos a la pestaña que dice “cuadro de herramientas”.

• Arrastramos un “Label” hacia el formulario para las Etiquetas.

• Arrastramos dos “Buttons” hacia el formulario.

• Como se muestra a continuación.

• Daremos clic derecho sobre el formulario y seleccionaremos ver código o en su caso daremos F7.

• Agregaremos las librerías de C#.

• Using “System.Drawing” esto es para que nos permite ejecutar correctamente las líneas dibujadas.

• Using “System.Threading” esto es para que los hilos se ejecuten correctamente.

• A continuación, le agregaremos el código al evento de “Load” creamos una variable Random y la ejecutamos dentro del evento.

• A continuación, le daremos clic al botón” Círculos Rojos” esto para que nos cree un método clic.

• Se crea el hilo.

• Creamos un for para la cantidad de círculos que se nos van a dibujar en este caso 10.

• Mandamos las posiciones para que se dibujen en “Width e Heigth”.

• Dibujamos una elipse mandamos los parámetros de color, grosor, y las posiciones.

• Cada 100 milisegundos nos dibujaran los círculos.

• El hilo se inicia.

• A continuación, le daremos clic al botón” Círculos Azules” esto para que nos cree un método clic.

• Se crea el hilo.

• Creamos un for para la cantidad de círculos que se nos van a dibujar en este caso 100.

• Mandamos las posiciones para que se dibujen en “Width e Heigth”.

• Dibujamos una elipse mandamos los parámetros de color, grosor, y las posiciones.

• Cada 600 milisegundos nos dibujaran los círculos.

• El hilo se inicia.
