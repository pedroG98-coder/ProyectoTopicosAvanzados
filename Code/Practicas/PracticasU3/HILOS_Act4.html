Práctica:"Juego Carrera de Carros"

Solución de la Actividad 4

• Para comenzar con la actividad debemos:

• Abrir nuestro IDE de Visual Studio 2019.

• Crear el nuevo proyecto..

• Escoger la plantilla de “Aplicación de Windows Forms (.Net Framework).”.

• Dar clic en siguiente.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Agregaremos un nombre al proyecto.

• Agregaremos un nombre a la solución o esta a su vez puede ser la misma que la del proyecto.

• Clic en crear.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Posteriormente nos dirigimos a la pestaña que dice “cuadro de herramientas”.

• Arrastramos un “Label” hacia el formulario para las Etiquetas.

• Arrastramos el “PictureBox” hacia el formulario.

• Arrastramos un “Botón” hacia el formulario.

• Agregamos las imágenes que deseamos mostrar, en este caso será de carritos y banderas de meta.

• Como se muestra a continuación.

• Daremos clic derecho sobre el formulario y seleccionaremos ver código o en su caso daremos F7.

• Agregaremos las librerías de c#.

• Using “System.Drawing” esto es para que nos permite ejecutar correctamente las líneas dibujabas.

• Using “System.Threading” esto es para que los hilos se ejecuten correctamente.

• Seleccionaremos el botón o la propiedad que dice “Eventos” el que tiene la forma de un rayo.

• De ahí se despliega una lista de eventos, de la cual escogeremos el “Paint”.

• Damos clic a cada evento para que nos cree el código.

• Como se muestra a continuación.

• Posteriormente le agregaremos el código al evento de “Paint” esto es para que en el formulario al iniciar la ejecución del programa nos dibuje 3 líneas rectas, esto para simular como un tipo carril para el auto.

• Mandamos a traer las variables pen al cual le agregamos un color y un groso en sus parámetros.

• Dibujamos la línea en la posición que deseemos en el formulario dándole valores a los parámetros.

• Seleccionaremos el botón o la propiedad que dice “Eventos” el que tiene la forma de un rayo.

• De ahí se despliega una lista de eventos, de la cual escogeremos el “Load”.

• Damos clic a cada evento para que nos cree el código.

• Como se muestra a continuación.

• Posteriormente le agregaremos el código al evento de “Load” ejecutaremos los varios hilos, como se muestra a continuación.

• En seguida le agregaremos un delegado para los PictureBox, dándole como parámetros la posición y la velocidad.

• Se crea un método donde mandamos a traer el delegado creado anteriormente y con el parámetro mover.

• Hacemos la validación de los carritos para la posición, y la velocidad de cada uno, en este caso el auto uno ira más rápido, el segundo menos lento y el tercero más lento.

• A continuación, se crea otro método mover donde mandamos como parámetro el PictureBox, la posición y la velocidad.

• Se crea un for para el conteo del largo de la carrera hasta donde llegara la posición del formulario.

• Por último, se duerme el hilo y la velocidad.

• Por último, al darle clic al botón se crea otro método.

• Se crean 3 hilos.

• Se crea un for con el largo de los hilos, esto para que vaya aumentando.

• En el hilo se manda a traer como parámetro el método creado anteriormente.

• Se inician los hilos.
