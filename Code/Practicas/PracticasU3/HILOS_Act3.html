Práctica:"Juego Galaga"

Solución de la Actividad 1

• Tendremos que crear un nuevo proyecto de visual studio, escogemos la plantilla de “Aplicación de Windows Forms (.Net Framework)”.

• Una vez creado colocaremos los elementos necesarios para llevar a cabo esta práctica, los cuales son:

• 2 ProgressBar, uno para tu personaje y otro para el enemigo.

• 2 PictureBox, uno para tu personaje y otro para el enemigo.

• 1 FlowLayoutPanel.

• 2 label, para identificar la vida de cada personaje

• Recuerda que para agregar estos componentes tendrás que dirigirte al apartado de “Cuadro de herramientas.”

• Una vez colocados los elementos podrá modificar sus propiedades, tales que los Picturebox colocaremos imágenes para hacer más llamativo el juego, para eso daremos clic al elemento PictureBox y en el apartado de propiedades nos dirigiremos a la sección de imagen, daremos clic y seleccionaremos una imagen que deseamos poner, ejemplo:

• Lo mismo haremos para la nave enemiga, lo recomendable es colocar una imagen en formato png para que se vea lo mejor posible.

• También al formulario colocaremos un evento llamado KeyPress el cual nos servirá para poder mover el personaje dentro del formulario. Nos dirigiremos al apartado de propiedades y daremos clic al rayo, buscaremos el veneto KeyPress y daremos doble clic para crear el evento.

• Ahora procederemos a programar la lógica del juego, daremos clic derecho dentro del formulario y seleccionaremos ver código o en su defecto F7.

• Colocaremos algunas variables que nos servirán para guardar la posición del personaje y de la nave, la vida de los personajes, etc. También crearemos los hilos correspondientes para cada personaje para que puedan disparar, moverse y generar las balas.

• Crearemos un método el cual generara las balas automáticamente. Codigo correspondiente:

                
public void m_agregar(Control c)
            {
                     if (this.InvokeRequired)
                     {
                        d_agregar da = new d_agregar(m_agregar);
                        object[] pars = new object[] { c };
                        this.Invoke(da, pars);
                     }
                        else
                     {
                        if (c is PictureBox)
                     {
                        c.Size = new Size(10, 15);
                        c.BackColor = Color.Red;
                        this.Controls.Add(c);
            }
      }
}

• Dicho método permitirá generar las balas tonto del personaje como el de la nave enemiga, también generamos un picturBox el cual tiene como tamaño 1 0 y 15 y sera de color rojo.

• Ahora crearemos un método que hará desaparecer las balas al momento de desaparecer de la visión del jugador. Código correspondiente.

                
public void m_desapareser(Control c)
            {
                 if (this.InvokeRequired)
            {
                 d_desapareser dd = new d_desapareser(m_desapareser);
                 object[] pars = new object[] { c };
                 this.Invoke(dd, pars);
                    
            }
                 else
            {
                 c.Dispose();
    }
}

• Crearemos un método el cual permitirá que disparen los personajes. Código correspondiente.

                    
public void m_disparar(PictureBox p, int x, int y)
        {
            if (this.InvokeRequired)
        {
            d_disparar dd = new d_disparar(m_disparar);
            object[] pars = new object[] { p, x, y };
            this.Invoke(dd, pars);
        }
            else
        {
            p.Location = new Point(x, y);
    }
}

• Si pudiste darte cuenta los métodos son muy parecidos ya que lo que hacen es generar objetos y guardarlos en variables temporales para poder así utilizarlos en métodos que crearemos más adelante.

• El siguiente método es el que hará que nuestro personaje pueda disparar. Código correspondiente.

                        
public void disparar()
        {
            int x = this.pictureBox1.Location.X + (this.pictureBox1.Width / 2);
            int y = this.pictureBox1.Location.Y;
            PictureBox p = new PictureBox();
            m_agregar(p);
            while (y >= 0)
                {
                    m_disparar(p, x, y);
                    Thread.Sleep(20);
                    y -= 5;
                    if (p.Bounds.IntersectsWith(this.pictureBox2.Bounds))
                    {
                        VidadEnemigo();
                        break;
                    }
                }
                m_desapareser(p);
        } 
}

• Creamos variables que guarden la posición de nuestro personaje, mandamos a llamar el método m_agragar y dentro del ciclo llamamos el método m_disparar creados anteriormente, colocamos la velocidad a la cual deseamos que se mueva la bala y cada cuantos pixeles queremos que se mueva, también creamos una condición que al momento de que la bala choque con la nave enemiga llame a un método llamado vidadEnemigo para que baje su vida, por ultimo invocamos el método m_desaparecer que es el encargado de eliminar la bala para que no consuma mucha memoria el programa.

• El método_moverPersonaje indica cuantos pixeles se moverá el personaje cada que presionemos una tecla y de igual manera el método M_Mover_Enemigo hace lo mismo, solo que este se moverá de manera automática e indicamos cada cuanto tiempo queremos que se mueva.

• Por último, el evento antes creado llamado keyPress lo usaremos para colocar las letras con las cuales deseamos movernos y con cual tecla dispara. Código correspondiente:

                            
private void Form1_KeyPress(object sender, KeyPressEventArgs e)
            {
                if (mover)
                {
                    if (e.KeyChar == (char)Keys.Space)
                    {
                        Thread t = new Thread(disparar);
                        t.Start();
                    }
                    else
                    if (e.KeyChar == 'a')
                    {
                        MoDerecha = true;
                        Thread hiloPer = new Thread(metodo_moverPersonaje);
                        hiloPer.Start();
                    }
                        else if (e.KeyChar == 'd')
                    {
                        MoIzquierda = true;
                        hiloPer = new Thread(metodo_moverPersonaje);
                        hiloPer.Start();
                    }
        }   
}
