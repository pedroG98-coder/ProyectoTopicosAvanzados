Solución de la Actividad 1

• Para comenzar con la actividad debemos:

• Abrir nuestro IDE de MySQL Workbench.

• Clic derecho en espacio blanco y seleccionamos la opción de “Create Schema”.

• Se abrirá una nueva ventana donde daremos clic en “Apply”.



• Posteriormente nos dirigirá a otra ventana y solo daremos clic en "Finalizar”.

• Como podemos observar ya tenemos creada la base de datos “Login”.

• Posteriormente ahora crearemos una tabla dentro de la base de datos.

• Desplegaremos las opciones de la base de datos login.

• Dar clic derecho a la opción de “Tables”.

• Seleccionar la Opción de “Create Table”.

• Se abrirá una nueva ventana donde pondremos el nombre de la tabla en este caso “Usuario”.

• Agregaremos dos atributos más las cuales son “Usuario” y “Contraseña”.

• Arrastramos dos “Buttons” hacia el formulario.

• Clic en Apply.

• Se abrirá una ventana donde nos dará una vista de todo el script y lo que contendrá la base de datos.

• Clic en Apply.

• Si todo sale bien nos saldrá una ventana donde solo finalizaremos.

• Ahora desplegaremos la base de datos hasta llegar a columnas.

• En este caso yo, ya tengo registrado un usuario y una contraseña la cual ocuparemos para la práctica del login con Visual Studio c#.

• Ahora seguiremos trabajando con la parte del programa en Visual Studio C#.

• Para comenzar con la actividad debemos:

• Abrir nuestro IDE de Visual Studio 2019.

• Crear el nuevo proyecto..

• Escoger la plantilla de “Aplicación de Windows Forms (.Net Framework).”.

• Dar clic en siguiente.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Agregaremos un nombre al proyecto.

• Agregaremos un nombre a la solución o esta a su vez puede ser la misma que la del proyecto.

• Clic en crear.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Posteriormente nos dirigimos a la pestaña que dice “cuadro de herramientas”.

• Arrastramos 5 “Labels” hacia el formulario para las Etiquetas.

• Arrastramos dos “Textbox” hacia el formulario.

• Arrastramos un “Botón” hacia el formulario.

• Como se muestra a continuación.

• Enseguida vemos en nuestra solución que tenemos el proyecto del login.

• Le daremos clic derecho en la opción de “Referencias”.

• Seleccionaremos la opción de “Agregar Referencia”.

• Enseguida se abrirá una nueva ventana donde buscaremos la extensión.

• La extensión a buscar es la que se llama “MySQL.Data” y tiene que tener una palomita como se muestra a continuación.

• Como podemos observar se agregó correctamente la extensión de “MySQL.Data”.

• • Daremos clic derecho sobre el formulario y seleccionaremos ver código o en su caso daremos F7.

• Agregaremos la librería de conexión de “MySQL”.

• Using “MySql.Data.MySqlClient” esto es para que nos permite ejecutar correctamente la conexión con la base de datos.

• A continuación, le daremos clic al botón “Iniciar Sesión” esto para que nos cree un método clic.

• Abrimos la conexión y como parámetros, mandamos a traer en este caso localhost o puede ser mediante la IP= 127.0.0.01, el nombre de nuestra base de datos, el User, y la contraseña.

• Abrimos conectar.

• Posteriormente establecemos una conexión la base de datos MySQL.

• Obtenemos la instrucción Transact-Sql y como parámetros hacemos un Select con el nombre de la tabla para así ejecutar correctamente los datos de la tabla como son el usuario y la contraseña.

• Posteriormente hacemos una validación de que, si el nombre de usuario y la contraseña son correctos, no mandara un mensaje de “Bienvenido”.

• De lo contrario si los datos introducidos son incorrectos nos mandara un mensaje de “Nombre de Usuario o Contraseña Incorrecta”.

• Cerramos la conexión.
