Práctica:"Modulo Insertar (CRUD)"

Solución de la Actividad 3

• Para comenzar con la actividad debemos:

• Abrir nuestro IDE de SQL Server.

• Clic derecho en la carpeta de base de datos y seleccionamos la opción de “Nueva base de Datos”.

• Se abrirá una nueva ventana donde tendremos que poner el Nombre de nuestra a base de datos a crear, en este caso el mío se llamará: “CRUDPrueba”.

• Posteriormente le damos clic al botón de aceptar.

• Como podemos observar ya tenemos la base de datos creada.

• Desplegamos las opciones de la base de datos.

• En la carpeta de Tablas le daremos clic derecho y en la opción de “Nuevo” y escogeremos la opción de tabla.

• Posteriormente se abrirá una nueva ventana donde tendremos que poner los atributos que vamos a desear almacenar, así como el tipo de dato.

• Ponemos como llave primaria el Producto ID.

• Guardamos y nombramos a la tabla en este caso se llama “Productos.”

• Como podemos observar ya se nos creó la tabla de Productos.

• Ahora haremos un Select para corroborar que este la tabla correctamente.

• En este caso tendremos una tabla vacía, ahora lo que toca hacer es un programa para poder registrar datos y se guardar en esta base de datos.

• Para comenzar con la actividad debemos:

• Abrir nuestro IDE de Visual Studio 2019.

• Crear el nuevo proyecto..

• Escoger la plantilla de “Aplicación de Windows Forms (.Net Framework).”.

• Dar clic en siguiente.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Agregaremos un nombre al proyecto.

• Agregaremos un nombre a la solución o esta a su vez puede ser la misma que la del proyecto.

• Clic en crear.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Posteriormente nos dirigimos a la pestaña que dice “cuadro de herramientas”.

• Arrastramos 6 “Labels” hacia el formulario para las Etiquetas.

• Arrastramos 5 “Textbox” hacia el formulario.

• Arrastramos 1 DataGridView.

• arrastramos un “Botón”hacia el formulario.

• Como se muestra a continuación.

• Posteriormente nos dirigimos a la pestaña que dice “Explorador de Servidores”.

• Le damos clic en Conexiones de datos.

• Escogemos la opción de “Agregar Conexión”.

• Como se muestra a continuación.

• Se abrirá una nueva ventana donde tendremos que poner el nombre del servidor de nuestra base de datos.

• Seleccionamos el nombre de la base de datos en este caso la mía es “CRUDPrueba”.

• Clic en “Probar Conexión”.

• Como se muestra a continuación y todo sale bien le daremos clic en “Aceptar”.

• Daremos clic derecho sobre el formulario y seleccionaremos ver código o en su caso daremos F7.

• Agregaremos la librería de conexión de “SQL”.

• Using “System.Data.SqlCliente esto es para que nos permite ejecutar correctamente la conexión con la base de datos.

Cadena de Conexión SQL

• Posteriormente nos dirigimos a la pestaña que dice “Explorador de Servidores”.

• Le damos clic derecho en la base de datos de “CRUDPrueba”.

• Clic en la opción de “Propiedades”.

• En la opción de “Cadena de Conexión” tendremos que copiar todo, esto se debe que lo ocuparemos para hacer la conexión entre el programa y la base de datos.

• Daremos clic derecho sobre el formulario y seleccionaremos ver código o en su caso daremos F7.

• Abrimos la conexión y como parámetros, pegamos toda la “cadena de conexión” que copiamos anteriormente.

• Como se muestra a continuación.

• A continuación, le daremos clic al botón “Insertar” esto para que nos cree un método clic.

• Abrimos conectar.

• Obtenemos la instrucción Transact-Sql y como parámetros hacemos un insert into con el nombre de la tabla y evaluamos todos los campos y conectamos.

• Ejecutamos la instrucción y devolvemos el número de filas afectadas.

• Mandamos un mensaje de: “El Producto Insertado Correctamente”.

• Cerramos la conexión.

• Mostramos los datos en el datagridview.

• Posteriormente crearemos una función de “mostrarDatos”, esto es para que todos los datos que tengamos almacenados en nuestra base de datos se reflejen en un “DataGridView”.

• Abrimos conectar.

• Obtenemos la instrucción Transact-Sql y como parámetros hacemos un select con el nombre de la tabla y conectamos.

• Hacemos la conexión y llenamos todos los “DataSet”.

• Creamos un nuevo DataTable.

• Llenamos todo el nuevo Datatable.

• Mostramos todos los datos guardados en un “DataGridView”.

• Ejecutamos la instrucción y devolvemos el número de filas afectadas.

• Posteriormente crearemos un evento “Load” esto es para cada vez que ejecutemos el programa nos cargue todos los datos y nos muestre la información en el “DataGridView”.

• Solo mandamos a traer la función creada anteriormente “mostrarDatos”.

• Como se muestra a continuación.
