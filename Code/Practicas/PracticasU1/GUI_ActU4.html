Práctica:"Componentes Gráficos"

Solución de la Actividad 4

• Para comenzar con la actividad debemos:

• Le damos un nombre y después clic en crear.

• Abrir nuestro IDE de Visual Studio 2019..

• Crear el nuevo proyecto..

• Escoger la plantilla de “Aplicación de Windows Forms (.Net Framework)”.

• Dar clic en siguiente.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Agregaremos un nombre al proyecto.

• Agregaremos un nombre a la solución o esta a su vez puede ser la misma que la del proyecto.

• Clic en crear.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Posteriormente nos dirigimos a la pestaña que dice “cuadro de herramientas”.

• Arrastramos un “Label” hacia el formulario para las Etiquetas.

• Arrastramos el “ListBox” hacia el formulario.

• Arrastramos el “ComboBox” hacia el formulario.

• Arrastramos el “PictureBox” hacia el formulario.

• Arrastramos un “Botón” hacia el formulario.

• Como se muestra a continuación.

• Posteriormente agregaremos los datos tanto en el ComboBox y en el ListBox como se muestra a continuación.

• Dar clic en la parte del componente.

• Se abrirá una ventana donde escogeremos la opción de “Editar Elementos”.

• Se abrirá una nueva ventana donde podremos los datos que mostraremos.

• Advertencia: Cada etiqueta que pongamos en cada uno de los componentes tendrá que llamarse así cada imagen a cuál será referenciado.

• Como se muestra a continuación.



• Por último, daremos doble clic al componente del “Boton” y se abrira una nueva ventana:

• Se crea un evento “Clic”.

• Posteriormente le agregaremos el código el cualquier hará que muestre las imágenes cuando selecciones un dato en el “ComboBox” o en el “ListBox”.

• Pasamos lo que es el nombre del “PictureBox” = pictureBox1.

• ImageLocation: es de donde mandamos a traer la imagen y entre “” mandamos lo que es la ruta de la imagen. Así mismo también le agregamos el tipo de extensión que es la imagen.

• Como se muestra a continuación.
