Práctica:"Hola Mundo"

Solución de la Actividad 1

• Abrimos Visual Studio 2019.

• Se mostrará la siguiente interfaz.

• Daremos clic en nuevo proyecto.

• Aparecerá la siguiente interfaz.

• Buscaremos dentro de las opciones presentadas la aplicación de Windows Forms (.NET Framework).

• Daremos clic en siguiente y aparecerá la siguiente interfaz.

• Buscaremos dentro de las opciones presentadas la aplicación de Windows Forms (.NET Framework).

• Colocamos un nombre a nuestro proyecto y un nombre a la solución, en este caso dejaremos el nombre de la solución igual que el nombre del proyecto.

• Daremos clic en crear.

• Aparecerá la siguiente interfaz.

• Podemos notar nuestro proyecto en la parte derecha con los archivos que lo componen.

• En la parte izquierda podemos encontrar unas pestañas, que al momento de darles clic nos mostraran:

1. Explorador de servidores.

2. Cuadro de herramientas.

3. Origen de datos.

• Ahora daremos clic en cuadro de herramientas

• En este apartado encontraremos todos los elementos visuales para nuestra interfaz.

• Para colocarlos solo basta con arrastrarlos al apartado del Form.

• Arrastraremos el elemento llamado label.

• Se mirará de la siguiente manera.

• Podemos modificar las propiedades del elemento, dirigiéndonos a la parte inferior derecha.

• Una vez echo algunas modificaciones, procederemos a ejecutar nuestro proyecto dándole clic a la flechita verde encontrada en la parte superior o presionando F5.

• Aparecerá la siguiente interfaz.

• Listo has creado tu primera interfaz de usuario.

• Ahora estas listo para crear interfaces de usuarios de lo más atractivas posible.

• En el transcurso del manual podrás aprende a controlar algunos elementos del apartado de cuadro de herramientas.
