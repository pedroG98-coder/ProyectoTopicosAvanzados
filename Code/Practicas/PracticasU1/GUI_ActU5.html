Práctica:"Imagenes Aleatorios"

Solución de la Actividad 5

• Crear un nuevo proyecto de tipo Aplicación de Windows Form (.NET Framework).

• Dar clic en siguiente.

• Colocar un nombre al proyecto y dar en crear.

• Posteriormente nos dirigiremos al aparatado de cuadro de herramientas y arrastraremos al form los siguientes elementos.

• Arrastrar un “Button” al formulario.

• Arrastrar un “FlowLayautPanel” al formulario.

• Arrastrar un “picturebox” al formulario.

• Arrastrar un “openFileDialog” al formulario.

• Modificamos las propiedades de los elementos tales como el “name” y al botón le colocamos el siguiente texto “Agregar Imagen”.

• Para agregar el evento al botón le daremos clic al icono del rayo el cual contiene todos los eventos que se le pueden agregar a dicho elemento. En esta ocasión elegiremos el evento clic.

• Daremos doble clic al componente del “Botón” y se abrirá una nueva ventana:

• Posteriormente le agregaremos el siguiente código.

Random r = new Random();
cont = r.Next(1, 4);
string ruta = @"C:\Users\RAY-X\Downloads\imagenes\"+cont+".jpg";
PictureBox pb = new PictureBox();
pb.Image = Image.FromFile(ruta);
pb.SizeMode = PictureBoxSizeMode.StretchImage;
pb.Tag = "im" + cont;
pb.Click += CambiarImagen;
this.flowLayoutPanel1.Controls.Add(pb);

• Generamos un objeto de tipo random llamado r, el cual generara números aleatorios del 1 al 4.

• Creamos una variable de tipo string llamada ruta, a la cual guardaremos la ruta de donde ocuparemos las imágenes.

• Creamos otro objeto utilizando el elemento pictureBox, el cual nos dejara visualizar las imágenes.

• Colocamos el SizeMode para que se ajuste al tamaño que le indicamos.

• Agregamos otro evento llamado cambiar imagen.

• Y finalmente todo lo cargamos al elemento flowLayautPanel.

• Como puedes observar nos da error en la línea 31 ya que aún no emos creado el evento cambiar imagen.

• Para eso haremos el siguiente código.

private void CambiarImagen(object sender, EventArgs e)
{
PictureBox pb = new PictureBox();
pb = (PictureBox)sender;
if (this.openFileDialog1.ShowDialog() == DialogResult.OK)
{
pb.Image = Image.FromFile(this.openFileDialog1.FileName);
}
pb.SizeMode = PictureBoxSizeMode.StretchImage;
}

• Creamos un evento llamado CambiarImagen.

• Creamos un objeto con el elemento pictureBox.

• Creamos una condición utilizando el elemento openFileDialog que nos manda como comparación un ok.

• Dentro del if utilizamos de nuevo el elemento openFileDialog para poder seleccionar la imagen que desaemos cambiar.

• Y finalmente le damos formato a la imagen.

• Presionamos F5 para correr el programa.

• Al momento de presionar el botón nos aparecerán imágenes de la carpeta que colocamos anteriormente.

• Daremos clic a una imagen y nos aparecerá la síguete interfaz donde seleccionaremos la imagen nueva que deseamos remplazar.

• Seleccionamos una y le daremos a abrir, como podrán ver se ha remplazado la imagen exitosamente.
