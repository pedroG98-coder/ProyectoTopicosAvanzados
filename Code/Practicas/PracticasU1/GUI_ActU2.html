Práctica:"Cambio de Imagenes"

Solución de la Actividad 2

1. Para comenzar con la actividad debemos:

• Abrir nuestro IDE de Visual Studio 2019

• Crear el nuevo proyecto

• Escoger la plantilla de “Aplicación de Windows Forms (.Net Framework).”

• Dar clic en siguiente.

• Abrir nuestro IDE de Visual Studio 2019

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Agregaremos un nombre al proyecto.

• Agregaremos un nombre a la solución o esta a su vez puede ser la misma que la del proyecto.

• Clic en crear.

• Enseguida se abrirá una nueva interfaz como la siguiente:

• Posteriormente nos dirigimos a la pestaña que dice “Cuadro de Herramientas”.

• Arrastramos un “Label” hacia el formulario para las Etiquetas

• Arrastramos el “PictureBox” hacia el formulario.

• Arrastramos un “Botón” hacia el formulario.

• Como se muestra a continuación.

• Para cambiar el texto en el botón tenemos que hacer lo siguiente:

• Seleccionar el botón.

• En la ventana de propiedades buscar la opción que dice “Text”.

• Agregamos un texto a mostrar esto se puede hacer con todo el tipo de componentes que manejemos.

• Después de agregar todos los elementos, ahora seleccionaremos nuevamente el “Botón”, en la ventana de propiedades del botón.

• Seleccionaremos el botón o la propiedad que dice “Eventos” el que tiene la forma de un rayo.

• De ahí se despliega una lista de eventos, de la cual escogeremos el “MouseHover” y “MouseLeave”.

• Damos clic a cada evento para que nos cree el código.

• Como se muestra a continuación

• Posteriormente le agregaremos el código el cualquier hará que muestre las imágenes cuando pasemos el puntero del mouse sobre el botón.

• Pasamos lo que es el nombre del “PictureBox” = pictureBox1.

• ImageLocation: es de donde mandamos a traer la imagen y entre “” mandamos lo que es la ruta de la imagen.
