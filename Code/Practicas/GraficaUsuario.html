Objetivo General.

Aprender a crear interfases de usuario desde el software Visual Studio 2019, en el lenguaje de programación C#(Csharp).


Practicas Unidad 1(Interfaz Gráfica de Usuario).

1.1 Hola Mundo.

La práctica consiste en realizar un programa con una interfaz gráfica de usuario donde muestre el siguiente mensaje “Hola Mundo” con el lenguaje de C# (CSHARP).

Ver procedimiento


1.2 Cambio de Imagen.


La práctica consiste en realizar un programa donde se utilice los eventos “MouseHover y MouseLeave” en el lenguaje C#.

Ver procedimiento


1.3 Calcular Distancia de un Objecto.

La práctica consiste en realizar un programa con interfaz gráfica de usuario que calcule la distancia de un objeto con la siguiente formula: “d = v * t”.

Ver procedimiento

1.4 Componentes Gráficos.

La practica consiste en realizar un programa con interfaz gráfica de usuario donde ocupemos: “ListBox,ComboBox y PictureBox”.

Ver procedimiento



1.5 Imagenes Aleatorios.

La práctica consiste en realizar un programa que cargue imágenes Random dentro de una carpeta y una vez cargada la imagen puedas cambiarla por la misma.

Ver procedimiento
